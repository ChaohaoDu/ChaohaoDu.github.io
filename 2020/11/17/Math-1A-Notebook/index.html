##2.2 Definitions of Limits P61

    • Formal definition of limit:

        When we say $$\lim\limits_{x\rightarrow a}f(x)=L$$, we mean for every $\epsilon>0$, there exists a $\delta>0$ such that $|f(x)-L|<\epsilon$ whenever $|x-a|<\delta$.

    • Informally:

      This says if $L$ is the limit, then we can let $f(x)$ get as close to $L$ as we want by letting $x$ get close and close to $a$.

3.7 The Chain Rule P185

Suppose $y=f(u)$ is differentiable at $u=g(x)$ and $u=g(x)$ is differentiable at $x $.The composite function $y=f(g(x))$ is differentiable at $x$, and its derivative can be expressed in two equivalent ways.

  • Version 1 $\dfrac{d y}{d x}=\dfrac{d y}{d u} \cdot \dfrac{d u}{d x}$
  • Version 2 $\dfrac{d}{d x}(f(g(x)))=f^{\prime}(g(x)) \cdot g^{\prime}(x)$

  1. Absolue Max and Min:

  2. Define:

    Let $f$ be defined on a set $D$ (some domain) and let $c\in D$. Then $f(c)$ is

    • an abolute/global maximun if $f(c)\geq f(x)$ for all $x\in D$
    • an abolute/global minima if $f(c)\geq f(x)$ for all $x\in D$

  3. Extreme Value Theorem:

    • Need closed interval (end pts. must be included)
    • Fucn must be conts on that closed interval

  4. Where can abs. max and min occur?

    at end pt. or critical point.

  5. Local Max and Min Values:

    1. Define:

      $c$ is an interior pt of some interval $I$ on which $f$ is defined.

      • a local max if $f(c)\geq f(x)$ for all $x \in I$
      • a local min if $f(c)\leq f(x)$ for all $x \in I$

    2. Local Extreme Value Theorem:

      if $f$ has a local max or min value at $c$ and $f’(c)$ exists, then $f’(c)=0$

  6. Critical Point:

    An interior point $c$ of the **domain** of $f$ at which $f'(c)=0$ or $f'(c)$ is undefined.

##4.2 What Derivatives Tell Us P245

  1. Increasing and Dereasing Functions:

    1. Define:

      $x_1$ and $x_2$ are in $I$ and $x_2>x_1$

      • $f$ is increasing on $I$, if $f(x_2)>f(x_1)$.
      • $f$ is decreasing on $I$, if $f(x_2)<f(x_1)$

    2. Test:

      • if $f’(x)>0$ on an interval $I$, then $f$ is increasing.

      • if $f’(x)<0$ on an interval $I$, then $f$ is decreasing.

  2. Find the local max and min, using calculus:

    1. Find critical pits (where $f’=0$ or $f’$ undefined)
    2. Sign test in $f’$ (First Derivgatie Test):
      • $f’$ has a local max at $c$ if $f’$ changes sign from $+$ to $-$
      • $f’$ has a local min at $c$ if $f’$ changes sign from $-$ to $+$
      • $f$ has no local extreme value at $c$ if $f’$ is $+/-$ on both sides near $c$
    3. Conclusion

  3. One Local Extremum Implies Absolute Extremum:

    $f$ is continuous on an interval $I$ and $c\in I$
    • if a local max occurs at $c$, the $f(c)$ is the absolute max of $f$ on $I$
    • if a local min occurs at $c$, the $f(c)$ is the absolute min of $f$ on $I$

  4. If $f$ is continuous at $c$ and $f$ changes concavity at $c$ ($f’’$ changes sign at $c$), then $c$ is an inflection point

  5. Concavity:

    1. Ddfine:

      let $f$ be differentiable on an open interval $I$

      • $f$ is concave up if $f’$ is increasing on $I$
      • $f$ is concave down if $f’$ is decreasing on $I$

    2. Test:

      • if $f’’>0$ on $I$, then $f$ is concave up on $I$
      • if $f’’<0$ on $I$, then $f$ is concave down on $I$

  6. Second Derivative Test for Local Extrema:

    $f''$ is continuous on an **open interval** containing $c$, and $f'(c)=0$
    • If $f’’(c) >0$, then $f$ has a local min at $c$
    • If $f’’(c) <0$, then $f$ has a local max at $c$
    • If $f’’(c)=0$, then $f$ is inconclusive

4.6 Mean Value Theorem P290

  1. Rolle’s Theorem:

    Let $f$ be **continuous** on a closed interval $[a,b]$ and **differentiable** on $(a,b)$ with $f(a)=f(b)$. There is at least one point $c$ in $(a.b)$ such that $f'(c)=0$

  2. Mean Value Theorem:

    1. Define:

      If $f$ is continuous on the closed interval $[a,b]$ and differentiable on $(a,b)$, then there is at least one point in $(a,b)$ such that $\dfrac{f(b)-f(a)}{b-a}=f’(c)$

    2. Zero Derivative Implies Constant Function:

      if $f$ is differentiable and $f'(x)=0$ at all points of an interval $I$, then $f$ is a coinstant function on $I$
